Where the Money Came From vs. Where the Breakthrough Landed

Each flow traces funding sources through discovery mechanisms to outcome industries. The width of each band shows how many discoveries share that path.

The Knowledge Web

Each node is a domain of knowledge. Links connect origin domains to outcome domains — each link is a discovery that jumped between them. Thicker links mean more discoveries crossed that gap. The node color shifts from teal (mostly an origin) to purple (mostly a destination). Drag nodes to explore.

Who Hosts the Random Walk?

Each arc connects an institution type to an outcome industry. The height and thickness of the arc reflects how many random walks share that path, colored by mechanism.

Geography of Accidents

Each ribbon connects a country of origin to an outcome industry. The thickness represents the number of discoveries flowing through that connection.
